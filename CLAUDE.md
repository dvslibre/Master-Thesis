# Projekt: Master Thesis

## Struktur

- `Melli/` – Referenzcode (NICHT verändern, nur als Kontext lesen)
- `Meiner/` – Hier entsteht MEIN neuer Code

## Referenz-Ordner Melli/ enthält:

- `thesis-main/` – Haupt-Thesis-Code (Python: preprocessing, MLP, ct_segmentation)
- `thesis-NEUSTART_TV/` – TV-Regularisierung Variante
- `thesis-new-ray-TV-feiner-` – Verfeinerte Ray-TV Version
- `thesis-normalisierung-2.0/` – Normalisierungs-Experimente
- `thesis-rollback_priors/` – Prior-basierte Rekonstruktion
- `thesis-stable_reference_run/` – Stabiler Referenzlauf

## Regeln

- Neuer Code kommt IMMER in `Meiner/`
- Dateien in `Melli/` niemals verändern
- Python bevorzugen, Jupyter-kompatibel
